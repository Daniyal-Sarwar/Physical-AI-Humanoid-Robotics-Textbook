# Module 1: ROS 2 Examples

This directory contains downloadable code examples for Module 1: ROS 2 Fundamentals.

## Contents

- `minimal_publisher.py` - Basic ROS 2 publisher node
- `minimal_subscriber.py` - Basic ROS 2 subscriber node
- `minimal_service.py` - Basic ROS 2 service server
- `minimal_client.py` - Basic ROS 2 service client

## Prerequisites

- ROS 2 Humble or later
- Python 3.10+

## Usage

```bash
# Source ROS 2
source /opt/ros/humble/setup.bash

# Run publisher
python3 minimal_publisher.py

# In another terminal, run subscriber
python3 minimal_subscriber.py
```
