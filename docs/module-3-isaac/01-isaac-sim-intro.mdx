## What is Isaac Sim?

**NVIDIA Isaac Sim** is a robotics simulation platform built on Omniverse:

- **Photorealistic rendering**: RTX ray tracing for accurate visuals
- **Accurate physics**: PhysX 5 with GPU acceleration
- **Synthetic data**: Generate labeled training data at scale
- **ROS 2 integration**: Native support via Isaac ROS
- **Domain randomization**: Automatic variation for robust ML

### Isaac Sim vs. Gazebo

| Feature | Isaac Sim | Gazebo |
|---------|-----------|--------|
| Rendering | RTX ray tracing | OpenGL |
| Physics | PhysX 5 | ODE/Bullet/DART |
| Synthetic data | Built-in (Replicator) | Limited |
| ROS support | Isaac ROS bridge | ros_gz |
| GPU required | Yes (NVIDIA) | No |
| Cost | Free (non-commercial) | Free |

## Installation

### System Requirements

- **GPU**: NVIDIA RTX 2070+ (8GB VRAM minimum)
- **CPU**: Intel Core i7 / AMD Ryzen 7
- **RAM**: 32GB recommended
- **Storage**: 50GB+ SSD space
- **Driver**: NVIDIA 525.60.11+

### Installing Isaac Sim

```bash title="Terminal"
# Install NVIDIA Omniverse Launcher
# Download from: https://www.nvidia.com/en-us/omniverse/

# Install Isaac Sim via Omniverse Launcher
# Navigate to: Exchange → Isaac Sim → Install

# Verify installation
cd ~/.local/share/ov/pkg/isaac_sim-*/
./isaac-sim.sh --help
```

### First Launch

```bash title="Terminal"
# Launch Isaac Sim
~/.local/share/ov/pkg/isaac_sim-*/isaac-sim.sh

# Or with specific settings
./isaac-sim.sh --/renderer/enabled=rtx --/app/livestream/enabled=false
```

## USD: The Foundation

Isaac Sim uses **Universal Scene Description (USD)**:

```python title="Creating USD Stage"
from pxr import Usd, UsdGeom

# Create a new stage
stage = Usd.Stage.CreateNew("my_scene.usda")

# Add a cube
cube = UsdGeom.Cube.Define(stage, "/World/MyCube")
cube.GetSizeAttr().Set(1.0)

# Save
stage.Save()
```

### USD Advantages

- **Composition**: Layer scenes non-destructively
- **Variants**: Multiple configurations in one file
- **References**: Reuse assets across scenes
- **Collaboration**: Multiple users edit simultaneously

## Importing Robots

### URDF Import

```python title="Python Script"
import omni.isaac.core.utils.stage as stage_utils
from omni.isaac.urdf import _urdf

# Configure URDF importer
urdf_interface = _urdf.acquire_urdf_interface()
import_config = _urdf.ImportConfig()
import_config.merge_fixed_joints = True
import_config.fix_base = False
import_config.make_default_prim = True

# Import URDF
result = urdf_interface.parse_urdf(
    "/path/to/robot.urdf",
    import_config
)
urdf_interface.import_robot(
    "/path/to/robot.urdf",
    result,
    import_config,
    "/World/Robot"
)
```

## ROS 2 Integration

### Enabling ROS 2 Bridge

```python title="Isaac Sim Script"
import omni.graph.core as og
from omni.isaac.ros2_bridge import _ros2_bridge

# Enable ROS 2 extension
import omni.kit.app
manager = omni.kit.app.get_app().get_extension_manager()
manager.set_extension_enabled_immediate("omni.isaac.ros2_bridge", True)
```

### Publishing Robot State

```python title="Joint State Publisher"
# Create OmniGraph for ROS 2 publishing
keys = og.Controller.Keys
og.Controller.edit(
    {"graph_path": "/ROS2Graph"},
    {
        keys.CREATE_NODES: [
            ("OnPlaybackTick", "omni.graph.action.OnPlaybackTick"),
            ("JointStatePublisher", "omni.isaac.ros2_bridge.ROS2PublishJointState"),
        ],
        keys.CONNECT: [
            ("OnPlaybackTick.outputs:tick", "JointStatePublisher.inputs:execIn"),
        ],
        keys.SET_VALUES: [
            ("JointStatePublisher.inputs:topicName", "/joint_states"),
            ("JointStatePublisher.inputs:targetPrim", "/World/Robot"),
        ],
    }
)
```
